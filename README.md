# Humanoid Robot Dance – Webots Simulation

## Project Overview
This project implements a dance routine for the NAO humanoid robot using the Webots robotics simulator. The robot performs a sequence of five distinct movements, each involving multiple limbs simultaneously, to create a complete choreographed dance routine.

## Robot Used
- **Robot:** NAO Humanoid Robot
- **Simulator:** Webots R2025a
- **Language:** Python 3

## Dance Movements
The routine includes the following 5 movements in order:

1. **Bow** – The robot lowers its head and brings both arms down as a greeting
2. **Wave** – The robot raises its right arm and waves side to side while turning its head
3. **Clap** – Both arms move inward and outward repeatedly to simulate clapping
4. **Head Groove** – The robot moves its head left and right while alternating arm positions
5. **Arm Raise** – Both arms are raised above the head and brought back down
